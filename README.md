# 工厂分布式智能巡检机器人系统
## Factory Distributed Intelligent Inspection Robot System

基于 Flask + WebSocket + Canvas 的智能巡检机器人监控管理平台。

### 功能模块

| 模块 | 说明 |
|------|------|
| 🗺️ 巡检地图 | 实时显示机器人位置、巡检点位、巡检路径 |
| 🤖 机器人管理 | 多机器人状态监控、远程控制（开始/停止/充电） |
| 📷 图像识别 | 上传设备图像，模拟深度学习识别（压板/仪表/红外） |
| 📊 SLAM建图 | 模拟栅格地图与激光扫描可视化 |
| 🔔 实时告警 | 异常检测自动告警、告警确认处理 |
| 📈 统计分析 | 巡检次数、故障趋势、电量统计图表 |
| 📡 物联网通信 | WebSocket 实时数据推送 |

### 快速运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动系统
python app.py

# 3. 打开浏览器访问
# http://localhost:5000
```

### 技术栈

- **后端**: Flask + Flask-SocketIO + Eventlet
- **前端**: Bootstrap 5 + Chart.js + Canvas API
- **通信**: WebSocket 实时推送
- **图像处理**: Pillow + NumPy

### API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/robots | 获取所有机器人状态 |
| GET | /api/robots/:id | 获取单个机器人详情 |
| POST | /api/robots/:id/command | 发送控制指令 |
| GET | /api/points | 获取巡检点位 |
| GET | /api/logs | 获取巡检记录 |
| GET | /api/alerts | 获取告警列表 |
| POST | /api/alerts/:id/acknowledge | 确认告警 |
| GET | /api/stats | 获取统计数据 |
| POST | /api/recognize | 图像识别 |
| GET | /api/slam/map | 获取SLAM地图数据 |

### 说明

本系统为毕业设计**软件仿真演示版本**，机器人运动、传感器数据、图像识别结果均为模拟生成，用于展示系统架构与功能设计。
