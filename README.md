# 工厂分布式智能巡检机器人系统
## Factory Distributed Intelligent Inspection Robot System

基于 Flask + WebSocket + Canvas 的智能巡检机器人监控管理平台，支持多机器人自动巡检、实时地图追踪、图像异常识别与告警管理。

### 系统截图

系统包含 **7 个功能区域** 的真实工厂 2D 平面图，机器人沿巡检路径自动移动，实时更新位置与状态。

### 功能模块

| 模块 | 说明 |
|------|------|
| 🗺️ 巡检地图 | 真实工厂 2D 平面图，实时显示机器人位置、巡检点位、A*寻路沿走廊移动，支持指北针与比例尺 |
| 🤖 机器人管理 | 多机器人状态监控（巡检/充电/空闲/返回），远程控制（开始/停止/充电），自动路径巡检 |
| 🎨 主题切换 | 经典风 / 暗黑工业风两套 UI，导航栏一键切换，自动记忆 |
| 📷 图像识别 | 上传设备图像，基于 OpenCV 检测（HOG行人检测 / Haar级联 / 颜色异常 / 轮廓识别 / 红外热成像模拟） |
| 📊 SLAM建图 | 模拟栅格地图与激光扫描可视化 |
| 🔔 实时告警 | 异常检测自动告警、告警确认处理 |
| 📈 统计分析 | 巡检次数、故障趋势、电量统计图表 |
| 📡 物联网通信 | WebSocket 实时数据推送（0.5s 更新频率） |

### 技术栈

- **后端**: Flask + Flask-SocketIO + Eventlet
- **前端**: Bootstrap 5 + Chart.js + Canvas API
- **通信**: WebSocket 实时推送（10Hz 模拟 + 2Hz 广播）
- **图像处理**: OpenCV + Pillow + NumPy

### 机器人巡检机制

- **A* 寻路**：机器人沿走廊移动，穿过门洞，不会穿墙
- **自动路径分配**: 每台机器人分配随机巡检路径，按顺序逐个访问点位
- **平滑移动**: 10Hz 模拟频率，沿路径拐点逐段移动
- **到站检测**: 生成巡检记录 + 随机异常检测
- **低电量返回**: 电池 < 20% 自动返回充电站
- **充满重启**: 充电至 100% 后自动分配新巡检任务
- **多机协作**: 3 台机器人独立运行，互不干扰

### API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/robots | 获取所有机器人状态（含实时坐标） |
| GET | /api/robots/:id | 获取单个机器人详情 |
| POST | /api/robots/:id/command | 发送控制指令（start_patrol / stop / charge） |
| GET | /api/points | 获取巡检点位 |
| GET | /api/logs | 获取巡检记录 |
| GET | /api/alerts | 获取告警列表 |
| POST | /api/alerts/:id/acknowledge | 确认告警 |
| GET | /api/stats | 获取统计数据（含历史趋势） |
| POST | /api/recognize | 图像识别（支持上传图片） |
| GET | /api/slam/map | 获取 SLAM 栅格地图数据 |
